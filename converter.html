    Details

    Currency Converter. That name sounds like - well, it's all about money. This project is from the ALC(Andela Learning Community) program. We were asked to build this app using the currencyconverter API. You can see more about these by clicking the link below.

    Anyways, the features of the app includes: It works offline, by the help of Service Worker, Cache API and IndexedDB. The app could also be added to the user's homescreen with it's own icon -just like any other native mobile app. I have to tell the truth though, the layout of the app is not so cool.
